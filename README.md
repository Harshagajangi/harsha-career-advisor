# Harsha Career Advisor 🌟

Harsha Career Advisor is a simple and modern career guidance website designed to help students choose the right career path based on their interests, education, and goals.

The platform provides career guidance for students after 10th, 12th, Degree, and Engineering. It also includes a career interest test and consultation booking system to help students get personalized advice.

## 🚀 Features

- Career guidance for multiple education levels
- Career interest test using JavaScript
- Consultation booking through Google Forms
- Student success stories
- Mobile-friendly responsive design
- Simple and clean user interface

## 🧑‍🎓 Target Users

- Students after 10th
- Students after 12th
- Engineering and degree students
- Students preparing for government or defence exams

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Forms (for consultation booking)
